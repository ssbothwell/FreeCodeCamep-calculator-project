Free Code Camp Calculator Project by Solomon Bothwell

This calculator is modeled after the MacOS calculator.app. It has operation chaining which can be achieved by hitting the calculate multiple times and by contining to enter operators instead of pressing the calculate button. The core calculator script was written using closures in calculator.js and rendering/function mapping to buttons is done in script.js
